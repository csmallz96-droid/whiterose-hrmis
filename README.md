# Whiterose HRMIS

**Human Resource Management Information System**  
Whiterose Venyou Enterprises Limited  
Mombasa, Kenya

## About
Whiterose HRMIS is the internal HR and payroll management system for Whiterose Venyou Enterprises Limited. It manages payroll, leave, contracts, performance, and employee records for 68 staff across 11 branches nationwide.

## Modules
- Dashboard
- Employees
- Payroll (KRA-compliant PAYE, NSSF, SHA, AHL, NITA)
- Payslips
- Leave Management
- Contracts
- Performance Appraisal
- Employee Self-Service
- Reports
- Onboarding
- Expenses
- Settings

## Tech Stack
- React + TypeScript + Vite
- Supabase (PostgreSQL + Auth)
- Tailwind CSS
- Deployed on Render
